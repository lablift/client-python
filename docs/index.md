# Introdução

O cliente Python da LabLift foi desenvolvido para facilitar a integração com nossos produtos. Nesta página, encontrará instruções básicas de uso.

## Pré-requisitos

Python 3.8+

## Instalação

```console
pip3 install lablift_client
```
